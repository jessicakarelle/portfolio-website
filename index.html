Survie en terre inconnue
Description

Jeu d'aventure console développé en C# dans le cadre d'un projet académique. Le joueur incarne un survivant dans un territoire inconnu et doit gérer ses ressources, explorer la carte et crafter des objets pour survivre. Le jeu utilise une architecture modulaire avec séparation des responsabilités, facilitant l'ajout de nouvelles fonctionnalités. Système de sauvegarde en JSON permettant de reprendre la partie à tout moment.

Fonctionnalités
  • Système de sauvegarde JSON
  • Gestion d'inventaire
  • Système de crafting
  • Progression temporelle
